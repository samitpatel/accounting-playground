# Quick Start Guide

Get started with Business Transaction Manager in 5 minutes!

## 1. Open the Application

Simply open `index.html` in your web browser. No installation required!

## 2. Create Your Account

1. Click **"Register"** on the login screen
2. Enter a username (minimum 3 characters)
3. Enter a password (minimum 6 characters)
4. Confirm your password
5. Click **"Register"**

## 3. Login

1. Click **"Login"** after registration
2. Enter your username and password

## 8. Add New Transaction

Click **"+ Add Transaction"** button (or the **+** floating button on mobile):

1. Select **Type**: Income or Expense
2. Enter **Amount**: e.g., 1500.00
3. Choose **Category**: From dropdown (changes based on type)
4. Pick **Date**: Transaction date
5. Add **Description**: Optional details
6. Add **Tags**: Comma-separated (optional)
7. Click **"Save Transaction"**

## 9. Export Data

1. Go to **Transactions** tab
2. Optionally filter data first
3. Click **"Export CSV"**
4. File downloads automatically as `transactions_YYYY-MM-DD.csv`
5. Open in Excel, Google Sheets, or any spreadsheet app

## 10. Switch Users

Want to try a different account?

1. Click **"Logout"** button (top right)
2. Confirm logout
3. Create a new account or login with different credentials
4. Each user gets their own companies and data!

## Common Tasks

### View Specific Month's Transactions
1. Go to Transactions tab
2. Set start date: 2025-09-01
3. Set end date: 2025-09-30
4. View September transactions

### Find All Marketing Expenses
1. Go to Transactions tab
2. Type "marketing" in search box
3. Or select "Marketing" from category filter

### Compare Two Companies
1. Open Company A (e.g., Tech Innovations)
2. Note dashboard numbers
3. Click company selector
4. Switch to Company B (e.g., Green Energy)
5. Compare dashboard numbers

### Add Monthly Rent
1. Click Add Transaction
2. Type: Expense
3. Amount: 1200
4. Category: Rent
5. Date: First of month
6. Description: Office rent
7. Tags: monthly, fixed
8. Save
